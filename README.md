# ShopApple
  Online shop technic apple  

 Это магазин техники Apple который создаетсья в учебном процессе  
 Этот сайт нужен для того что бы пользователи могли вырать товар онлайн и заказать его а так же посмотреть есть ли рядом с ними магизин техники Apple 

 Описание:
   Проект создается в учебных целях и прокачивание своих навыков  
     Сайт предназначен для покупки техники онлайн , основные функции на сайте это будет диаграмма с колличеством покупок некого товара , а так же раелизация карты , что бы пользователь  мог посмотреть 
     Ближайшие к нему иагазины Apple , которые будут отмечены на карте 
     
          npm create vite@latest
Это команда для установки React (на котором будет реализован сайт)

Для того что бы запустить этот проект нам нужно установить зависимость в наш React проект 

        npm i // npm instal
Писать можно как кому угодно с полным словом (установить ) или сокращенно (i)
